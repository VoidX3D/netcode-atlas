Cascading Style Sheets (CSS)

The language used to style and layout web pages

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

CSS Box Model

Every element in web design is a rectangular box. The CSS box model describes how these boxes are sized and spaced:

Box Model Components

  • Content - The actual content of the box
  • Padding - Clears an area around the content
  • Border - A border that goes around the padding
  • Margin - Clears an area outside the border

Box Sizing

The box-sizing property controls how the box model is calculated:

/* Traditional box model */
box-sizing: content-box; /* width = content width */

/* Modern box model */
box-sizing: border-box; /* width = content + padding + border */

CSS Layout Systems

Flexbox

One-dimensional layout model for arranging items in rows or columns.

.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

CSS Grid

Two-dimensional layout system for designing complex web layouts.

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 20px;
}

Positioning

Control the position of elements using static, relative, absolute, fixed, or sticky positioning.

.element {
  position: relative;
  top: 20px;
  left: 30px;
}

Floats

Legacy technique for wrapping text around images or creating layouts.

.image {
  float: left;
  margin-right: 20px;
}

CSS Selectors and Specificity

Common Selectors

  • element - Selects all elements of the specified type
  • .class - Selects elements with the specified class
  • #id - Selects the element with the specified ID
  • [attribute] - Selects elements with the specified attribute
  • :pseudo-class - Selects elements in a specific state
  • ::pseudo-element - Styles specific parts of an element

Specificity Hierarchy

When multiple rules apply to an element, specificity determines which rule takes precedence:

  1. Inline styles (1000 points)
  2. IDs (100 points)
  3. Classes, attributes, pseudo-classes (10 points)
  4. Elements, pseudo-elements (1 point)

The rule with the highest specificity wins. Use !important sparingly as it overrides all other rules.

Responsive Design

Media Queries

Apply different styles based on device characteristics like screen width, height, or orientation.

@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}

@media (min-width: 1200px) {
  .container {
    max-width: 1140px;
  }
}

Responsive Units

  • % - Percentage of parent element
  • vw/vh - Percentage of viewport width/height
  • rem - Relative to root element font size
  • em - Relative to parent element font size
  • fr - Fractional units in CSS Grid

CSS Variables and Custom Properties

Defining Variables

CSS custom properties (variables) allow you to store specific values to reuse throughout a document.

:root {
  --primary-color: #00d4ff;
  --secondary-color: #ff4df4;
  --spacing: 1rem;
  --border-radius: 0.75rem;
}

Using Variables

Reference variables using the var() function anywhere in your CSS.

.button {
  background-color: var(--primary-color);
  padding: var(--spacing);
  border-radius: var(--border-radius);
}

.card {
  border: 1px solid var(--secondary-color);
  margin-bottom: calc(var(--spacing) * 2);
}

CSS Animations and Transitions

Transitions

Gradually change property values over a specified duration.

.element {
  transition: all 0.3s ease;
}

.element:hover {
  transform: scale(1.05);
  box-shadow: 0 5px 15px rgba(0,0,0,0.2);
}

Animations

Create complex animations using keyframes.

@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

.element {
  animation: fadeIn 0.5s ease forwards;
}
